# feeding-the-few-pakistan-food-insecurity
Data science investigation into food insecurity across 16 Pakistani cities using SPI price data, FAO nutrition data, and World Bank indicators. Applied Linear Regression (R²=0.957), Gaussian Naive Bayes, and K-Means Clustering to identify highest-risk cities. Built an interactive 5-page Power BI dashboard. DS110 — Air University, Islamabad.
